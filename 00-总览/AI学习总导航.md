# AI 学习总导航

## 当前目标

把零散的 AI 输入，逐步沉淀成以下 4 层能力：

1. 概念理解
2. 技术判断
3. 动手实践
4. 体系输出

## 学习主线

建议按这 8 个模块持续积累：

1. 数学与基础直觉
   - 向量、矩阵、概率、优化
   - 损失函数、梯度下降、过拟合
2. 机器学习基础
   - 监督学习、无监督学习、训练/验证/测试
   - 特征、泛化、偏差方差、评估指标
3. 深度学习基础
   - 神经网络、反向传播、CNN、RNN、Transformer
4. 大模型原理
   - 预训练、指令微调、对齐、推理、上下文窗口
   - Token、Embedding、Attention、MoE、RAG、Agent
5. AI 工程化
   - Prompt、工作流、评测、缓存、向量库、部署
6. 多模态与 Agent
   - 图像、音频、视频、工具调用、规划执行
7. 商业与产品应用
   - AI 产品形态、场景设计、ROI、成本与风险
8. 安全与边界
   - 幻觉、隐私、版权、安全、评估、可控性

## 笔记处理流

每条新输入默认经过下面流程：

1. 归类：它属于哪个主题
2. 提炼：一句话说清重点
3. 展开：补充必要背景
4. 连接：关联已有概念、技术、案例
5. 落地：能做什么实践
6. 复习：生成复习问题或记忆点

## 输出物标准

后续我帮你整理内容时，优先产出这几类笔记：

- 主题笔记：讲清一个领域的完整框架
- 概念卡片：讲清一个术语到底是什么
- 资料摘录：保留来源和关键观点
- 实践笔记：记录怎么做、踩坑和结论
- 复盘笔记：记录哪些内容真正内化了

## 近期优先级

如果你没有特别指定主题，默认按下面顺序优先补齐：

1. Transformer 与 LLM 基础
2. Prompt / RAG / Agent 的关系
3. 模型评测与效果判断
4. AI 应用架构与工程实践
5. 商业落地案例
