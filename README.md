# AI 学习笔记系统

这套笔记系统是为“输入零散、持续积累、逐步系统化”设计的。

你的使用方式很简单：

1. 平时把你看到的零散内容直接发给我。
2. 我负责帮你做这几件事：
   - 判断它属于哪个主题
   - 提炼核心知识点
   - 补充上下文和关联概念
   - 放进合适的位置
   - 必要时生成复习题、行动项和实践建议
3. 你只需要持续输入，不需要先自己分类。

## 目录结构

- `00-总览/`
  - 学习总导航、路线图、方法说明
- `01-收件箱/`
  - 所有暂未整理的原始输入
- `02-主题/`
  - 按主题组织的系统化笔记
- `03-概念卡片/`
  - 单个概念的原子化笔记，适合复习和串联
- `04-论文与资料/`
  - 论文、文章、课程、工具资料摘录
- `05-项目实践/`
  - Demo、实验、代码实践、落地方案
- `06-复盘与输出/`
  - 周复盘、知识输出、问题清单
- `99-模板/`
  - 固定模板，方便后续持续追加

## 工作原则

这套系统遵循 5 个原则：

1. 先捕获，后整理
2. 先理解，后记忆
3. 先连接，后扩展
4. 以应用驱动学习
5. 用周期复盘抵抗遗忘

## 你后续怎么发内容

你可以直接发任意一种：

- 一段文字
- 一篇文章链接
- 一张图里的内容
- 一个概念名
- 一段代码
- 一个问题
- 一段你没看懂的话

我会自动帮你处理成笔记，而不是只做临时回答。

## 推荐节奏

- 平时：随手发内容给我
- 每周：我帮你做一次结构化整理
- 每两周：我帮你做一次知识网络补全
- 每月：我帮你做一次阶段复盘，更新下一阶段重点
